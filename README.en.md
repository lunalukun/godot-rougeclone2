# Rogue Mobile (Godot)

This project is a mobile-port effort based on Rogue2.Official (rougeclone2),
with a focus on comfortable portrait-mode play on smartphones.

Japanese version: [README.md](README.md)

## Overview

- Base: Rogue2.Official (rougeclone2)
- Goal: A portrait-oriented gameplay experience with touch-first controls
- Policy: Preserve core Rogue gameplay while optimizing UI and controls for mobile

## Development Environment

- Godot: 4.6
